# Dope Website Deployment Guide

## Option 1: Deploy to Netlify (Recommended - Free)

### Steps:
1. **Create a GitHub account** (if you don't have one)
   - Go to: https://github.com

2. **Upload your website files to GitHub**
   - Create a new repository called "dope-website"
   - Upload these files: index.html, style.css, script.js

3. **Connect to Netlify**
   - Go to: https://app.netlify.com
   - Click "Add new site" → "Import an existing project"
   - Select GitHub and choose your "dope-website" repository
   - Netlify will automatically detect it's a static site
   - Click "Deploy"

4. **Your site will be live at:**
   - Something like: `https://dope-website.netlify.app`

5. **(Optional) Add custom domain**
   - Go to "Domain settings" in Netlify
   - Add your custom domain (e.g., dopeautomation.com)

---

## Option 2: Deploy to Vercel (Also Free)

### Steps:
1. Go to: https://vercel.com
2. Sign up with GitHub
3. Click "Add New Project" → "Import Project"
4. Select your GitHub repository
5. Click "Deploy"

---

## Option 3: Manual Upload to Netlify (No Git)

If you don't want to use GitHub:

1. Go to: https://app.netlify.com
2. Click "Add new site" → "Deploy manually"
3. Drag and drop your website folder
4. Your site will be deployed instantly!
